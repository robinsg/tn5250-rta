# TN5250-RT

A modern, real-time terminal emulator for IBM 5250 protocol communication. This project provides a robust solution for connecting to and interacting with IBM i (formerly iSeries/AS400) systems through the TN5250 protocol.

## Overview

TN5250-RT is designed to deliver high-performance, feature-rich terminal emulation with real-time capabilities. It combines the reliability of traditional 5250 terminal functionality with modern development practices and real-time communication features.

### Key Features

- **TN5250 Protocol Support**: Full implementation of the IBM 5250 terminal protocol
- **Real-time Communication**: Optimized for low-latency, responsive interactions
- **Modern Architecture**: Built with contemporary software engineering practices
- **Cross-platform Compatibility**: Designed to work across different operating systems
- **Extensible Design**: Modular architecture for easy customization and extension

## Architecture

TN5250-RT follows a modular architecture with clear separation of concerns:

- **Protocol Layer**: TN5250 protocol implementation and handling
- **Terminal Emulator**: Display rendering and user input handling
- **Network Layer**: Connection management and data transmission
- **Configuration**: Runtime configuration and settings management

For detailed architecture documentation, see the [docs/](./docs/) folder.
